pBlog – The Next Generation

Posted on June 16, 2005 by jim Posted in Uncategorized — No Comments ↓

It’s been some time since I last blogged. Partially this is down to a lot of work and travel in China working with some very interesting clients on large-scale Web Services applications. Needless to say the MESTian philosophy prevails in those engagements, and really the only thing left is widespread adoption of SSDL :-) 

The other reason I haven’t blogged over the last couple of weeks is that Savas has been busy in the background migrating to me to a new version of pBlog that he’s been running for a few months now. He’s given me some interesting new client tools and has beefed up the Web controls that pBlog supports. Since it’s been around 12 months since I last submitted any code to pBlog I’ve been playing with the new toys locally.

So I’m now back to normal (if such a thing could ever have been said of me in the first place). Big thanks to the Savster for the migration work – beers on me the next time we catch up.
